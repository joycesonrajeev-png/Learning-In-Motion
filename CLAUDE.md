# Brain Puzzle Games

## Plugins

This repository is configured with the [Superpowers plugin](https://github.com/obra/superpowers), which provides an agentic skills framework and software development methodology for Claude Code.

The plugin is configured in `.claude/plugins.json` and includes composable skills for:
- Test-driven development (TDD)
- Systematic debugging
- Brainstorming and requirements refinement
- Subagent-driven development with code review
- Custom skill authoring
